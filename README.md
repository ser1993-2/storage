## Установка

- git clone https://github.com/ser1993-2/storage.git
- composer install
- php artisan:key:generate
- php artisan migrate

## Основные разделы

-  [GET] / - домашняя страница
-  [GET] /category - раздел категорий с CRUD
-  [GET] /product - раздел продуктов с CRUD 
-  [GET] /generate - генератор пароля и пин
